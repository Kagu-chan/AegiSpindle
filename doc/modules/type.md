Spindle.Type

Type utilities

* Shortname: Type
* Version: 1.0
* Author: Kagu-chan

###Spindle.type.isnil(mixed x)

###Spindle.type.isbool(mixed x)

###Spindle.type.isuserdata(mixed x)
