Spindle.CMD
-----------
Spindle command line integration

* Shortname: CMD
* Version: 1.0
* Author: Kagu-chan
* Source: [aegi-spindle-cmd.lua](https://github.com/Kagurame/AegiSpindle/tree/dev/src/aegi-spindle-cmd.lua)

> This module provides various functions to integrate command line interactions between LUA scripts and the whole system

###Spindle.cmd.getArgumentsTable(...)
Map given arguments to a table with named keys


###Spindle.cmd.execute(string command)
Executes given string command on command line and returns stdout
